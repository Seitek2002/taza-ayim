# Переключатель языка (KY/RU)

В проект добавлен простой клиентский i18n без сторонних библиотек. Переключатель находится в Header на главной странице.

## Где лежат переводы

- Файл словарей: `src/app/i18n/dictionaries.ts`
- Доступные языки: `ru` и `ky`

Структура словаря:

```ts
export const dictionaries = {
  ru: {
    nav: {
      about: 'О нас',
      services: 'Услуги',
      advantages: 'Преимущества',
      contacts: 'Контакты',
    },
    header: {
      contact: 'Связаться',
      whatsapp: 'Написать в WhatsApp',
    },
    brand: {
      subtitle: 'КЛИНИНГовая компания',
    },
  },
  ky: {
    /* ... */
  },
} as const;
```

### SEO-переводы

В словаре добавлены секции `seo` для RU и KY. Клиентский компонент `SeoClient` (см. `src/app/components/SeoClient.tsx`) обновляет:

- `<title>`
- `<meta name="description">`, `<meta name="keywords">`
- OpenGraph (`og:title`, `og:description`, `og:image`)
- Twitter (`twitter:title`, `twitter:description`, `twitter:image`)
- JSON-LD (`LocalBusiness`)

Компонент подключён в `app/layout.tsx` и реагирует на смену языка так же, как и вся страница.

## Как работает переключение

- Состояние и логика переключателя находятся в `src/app/page.tsx` внутри компонента `Header`.
- Выбранный язык хранится в `localStorage` по ключу `lang` и применяется при загрузке страницы.
- Атрибут `<html lang="...">` обновляется на клиенте согласно выбранному языку.

## Как добавить/изменить переводы

1. Откройте `src/app/i18n/dictionaries.ts` и добавьте/измените строки в соответствующих секциях (`nav`, `header`, `brand`).
2. Используйте значения из словаря в компонентах. В Header уже подключены:
   - Навигация (`nav.about`, `nav.services`, `nav.advantages`, `nav.contacts`)
   - Кнопки в шапке и мобильном меню (`header.contact`, `header.whatsapp`)
   - Подзаголовок бренда (`brand.subtitle`)
3. При необходимости подключайте словарь аналогично в других компонентах.

## Изменение языка по умолчанию

По умолчанию язык `ky`. Можно изменить дефолт в `useState<Lang>(() => ...)` в `Header` или предварительно записать `localStorage.setItem('lang', 'ru')`.
